## Objetivo

Construir una herramienta de línea de comandos en C++ que procese un dataset real de ventas retail, demostrando el uso de E/S de archivos, contenedores hash y parseo de argumentos con una librería profesional.

## Herramientas Utilizadas

| Herramienta | Propósito |
|-------------|-----------|
| `std::ifstream` | Leer el archivo CSV línea por línea |
| `std::stringstream` | Parsear cada línea separada por comas |
| `std::unordered_map` | Agregar transacciones por país en O(1) amortizado |
| `CLI11` | Parsear argumentos de la línea de comandos con flags y opciones |
| Structured bindings | Iterar el map con `auto& [pais, total]` |

## Dataset

El dataset (`Online Retail.csv`) contiene transacciones de una tienda en línea del Reino Unido con columnas:
`InvoiceNo, StockCode, Description, Quantity, InvoiceDate, UnitPrice, CustomerID, Country`

## Arquitectura del Programa

### Paso 1: Leer el CSV

```cpp
std::vector<Transaction> leerCSV(const std::string& archivo) {
    std::vector<Transaction> datos;
    std::ifstream file(archivo);
    std::string linea;

    std::getline(file, linea); // Saltar el header

    while (std::getline(file, linea)) {
        std::stringstream ss(linea);
        std::string token;
        std::vector<std::string> row;

        while (std::getline(ss, token, ',')) {
            row.push_back(token);
        }

        if (row.size() >= 8) {
            try {
                datos.push_back({
                    row[7],              // Country
                    std::stoi(row[3]),   // Quantity
                    std::stod(row[5])    // UnitPrice
                });
            } catch (...) { continue; } // Ignorar filas con datos corruptos
        }
    }
    return datos;
}
```

### Paso 2: Agregar Transacciones por País

```cpp
void transaccionesPorPais(const std::vector<Transaction>& datos,
                           const std::string& filtroPais = "") {
    std::unordered_map<std::string, int> conteo;

    for (const auto& t : datos) {
        if (filtroPais.empty() || t.country == filtroPais) {
            conteo[t.country]++;
        }
    }

    // C++17 Structured Bindings
    for (const auto& [pais, total] : conteo) {
        std::cout << std::left << std::setw(20) << pais
                  << ": " << total << " transacciones\n";
    }
}
```

### Paso 3: Interfaz CLI con CLI11

```cpp
int main(int argc, char** argv) {
    CLI::App app{"Analizador de Ventas Retail"};

    std::string pais;
    bool soloUK    = false;
    bool verTotal  = false;

    app.add_option("--country", pais,     "Filtrar por país");
    app.add_flag("--only-uk",   soloUK,   "Solo UK para el total");
    app.add_flag("--total",     verTotal, "Mostrar monto total de ventas");
```

## Conceptos Avanzados Aplicados

- **RAII con `std::ifstream`**: el archivo se cierra automáticamente al salir del scope, sin necesidad de llamar a `file.close()`.
- **`std::unordered_map` vs `std::map`**: el primero usa una tabla hash con O(1) promedio para inserción/búsqueda, ideal para grandes datasets.
- **Manejo robusto de errores**: bloques `try/catch` en el parseo para tolerar filas corruptas o con datos faltantes.
